serv00部署一个属于自己的alist云盘或者html网页教程

访问https://www.serv00.com进行注册您的帐号，有效期10年的服务器

1、注册好serv00的帐号后，连接您的帐号的SSH终端，可通过putty工具连接

复制的命令：cd ~/domains/*.*/public_nodejs

注释：上面的/*.*/位置是你部署的站点域名比如：cd ~/domains/pan.jinenyy.vip/public_nodejs 这样窗口就到了您的域名的public_nodejs路径如下面这个样子

[xxxxx@s3]:<~/domains/pan.jinenyy.vip/public_nodejs>$

2、下载serv00版的alist，默认下载最新版

复制的命令：bash <(curl -s https://jinenyy.vip/alist/serv00/install_alist.sh)

3、运行alist

复制的命令：./web.js server

4、运行后按ctrl键+c键停止并返回命令窗口

5、保活监听您的alist命令，让您的alist保持运行，不然会不定时后台进程就清理了导致您的alist无法访问

复制的命令：npm22 install

6、这时你不知道您的alist帐号和密码咋办，运行下面的命令即可，后面的set admin就是指设置的密码为admin的密码如果你要自己的密码那就在set 123456这样子就是你要设置的密码

复制的命令：./web.js admin set admin

7、要测试能不能保活您的alist的话关闭用户所有进程即可

复制的命令：killall -u $(whoami)
